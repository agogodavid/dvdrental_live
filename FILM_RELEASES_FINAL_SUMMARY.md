## Data Model

### Films Created
```sql
film (
  film_id,
  title,              -- Generated: "Mission: Tokyo"
  description,        -- Generated: "An elite agent must..."
  release_year,       -- Simulation year (2001, 2002, 2003)
  language_id,        -- English
  rental_duration,    -- 3 days
  rental_rate,        -- 20% of replacement_cost
  length,             -- Genre-appropriate (90-145 min)
  replacement_cost,   -- Genre-appropriate ($14-26)
  rating              -- Genre-appropriate (G-NC-17)
)
```

### Inventory Tracked
```sql
inventory (
  inventory_id,
  film_id,            -- Links to new film
  store_id,           -- All stores stocked
  date_purchased,     -- Release week
  staff_id            -- Who stocked it
)
```

### Categories Linked
```sql
film_category (
  film_id,            -- New film
  category_id         -- Matches template
)
```
